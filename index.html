HTML Tutorial

HTML is the standard markup language for Web pages.

With HTML you can create your own Website.

Example

<!DOCTYPE html>
<html>
  <head>
    <title>Page Title</title>
  </head>
  <body>
    <h1>This is a Heading</h1>
    <p>This is a paragraph.</p>
  </body>
</html>
