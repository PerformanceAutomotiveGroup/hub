# Performance Hub

An internal reference for web layouts, UI patterns, and
approved page compositions used across Performance Automotive Group projects.

## What This Is
- A learning and reference hub
- A source of approved layout patterns
- A place to document repeatable solutions

## What This Is Not
- A production codebase
- A live site framework
- A page builder

## Folder Overview
- /pages     → Documentation, examples, and templates
- /css       → Shared hub styles
- /js        → Shared hub scripts
- /nav       → Navigation partials
- /images    → Screenshots and visual references
