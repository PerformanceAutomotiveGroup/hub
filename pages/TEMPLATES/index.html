Template Library

This library contains reusable page templates created in WordPress and documented here for consistent reuse across Performance projects.

Templates are stored in GitHub as reference implementations. Always copy code from GitHub. Live WordPress pages may be linked for preview purposes only.

Page Layout Templates

Full-page compositions that define structure, content flow, and layout for common page types.

Examples: Landing pages, service pages, department pages

Section Templates

Reusable page sections designed to be dropped into multiple layouts without modification.

Examples: Hero sections, testimonials, feature grids, CTAs

Forms & Conversion Templates

Layouts focused on lead capture, user input, and conversion-driven interactions.

Examples: Contact forms, booking pages, quote requests

Content Templates

Structured layouts for editorial, informational, or long-form content.

Examples: Blog posts, FAQs, policy pages

Utility & Functional Templates

Pages that support tools, systems, or interactive functionality beyond standard content.

Examples: Location finders, calculators, dashboards

Campaign & Seasonal Templates

Promotional or time-bound layouts built for campaigns, launches, and events.

Examples: Sales events, recruitment drives, holidays

Framework-Specific Templates

Templates that depend on specific frameworks, plugins, or theme-level assumptions.

Examples: Bootstrap layouts, plugin-based pages
