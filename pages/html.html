HTML Basics

HTML stands for HyperText Markup Language. It's used to create the structure of web pages.

Example:

<!DOCTYPE html>
<html>
  <head>
    <title>My Page</title>
  </head>
  <body>
    <h1>Hello World!</h1>
    <p>This is a basic HTML page.</p>
  </body>
</html>
