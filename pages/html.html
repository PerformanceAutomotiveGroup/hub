HTML

What is HTML?

HTML stands for HyperText Markup Language. It is the standard language for creating webpages and web applications.

Basic Structure

An HTML document has a defined structure: doctype declaration, html, head, and body tags.

<!DOCTYPE html>
<html>
  <head>
    <title>Page Title</title>
  </head>
  <body>
    <h1>Welcome</h1>
  </body>
</html>
