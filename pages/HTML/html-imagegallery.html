HTML Image gallery

This HTML image map example displays an interactive image divided into five clickable areas. Each area highlights on hover and shows a label with its title.

/* Grid container */
.gallery-grid {
display: grid;
grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
gap: 20px;
margin: 20px 0;
}

/* Each gallery item */
.gallery-item {
border: 1px solid #ccc;
border-radius: 8px;
overflow: hidden;
background-color: #fff;
transition: transform 0.3s, box-shadow 0.3s, border-color 0.3s;
}

/* Hover effect */
.gallery-item:hover {
transform: translateY(-5px);
box-shadow: 0 10px 20px rgba(0,0,0,0.1);
border-color: #777;
}

/* Image inside gallery */
.gallery-item img {
width: 100%;
height: auto;
display: block;
transition: transform 0.3s;
}

/* Optional image zoom on hover */
.gallery-item:hover img {
transform: scale(1.05);
}

/* Description below each image */
.gallery-item .desc {
padding: 12px;
text-align: center;
font-size: 0.95rem;
color: #333;
}
